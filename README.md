This repository contains the appendices to our reproduction and robustness test report of the results published in:
> Brough, Jessica, Lasana T. Harris, Shi Hui Wu, Holly P. Branigan & Hugh Rabagliati. 2024. Cognitive causes of ‘like me’ race and gender biases in human language production. Nature Human Behaviour. Nature Publishing Group 8(9). 1706–1715. https://doi.org/10.1038/s41562-024-01943-3.

This repository contains a copy of the original paper (`paper_like_me.pdf`), as well as the authors' original scripts and datasets for ease of access (`R_scripts/scripts_and_data_original/`). The folder (`R_scripts/scripts_reproduction/`) contains our modified scripts, based on the authors' originals. The folder structure and file names follow the same nomenclature as the original files with the suffix `_reproduction`.

This reproduction was conducted as part of the [Replication Games 2025](https://indico.uni-muenster.de/event/3526/) hosted in Münster.
